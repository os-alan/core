State tab

This tab provides an editor for grid state.
  • The state is stored as a plain object
  • This object is used to set grid properties declaratively
  • Hypergrid has API calls for loading and saving state
  • Grid properties are generally own members of the grid.properties object (i.e., not including its prototype chain, which contains defaults, theming, etc.)
As with the Data tab editor, this editor also uses eval to parse straight JavaScript, not JSON. When editing you do not need to concern yourself with strict adherence to the JSON standard (double quotes, primitive types, _etc._).

Edit a grid property in the State tab

Show the grid row numbers:

  1. Switch to the State tab
  2. Edit the value of the value of the showRowNumbers property to be true rather than false
  3. Click the grid.setState(…) button
  4. The grid is now rendered with row numbers

Dynamic properties

Grid properties can also be dynamic (or "smart") which means they:
  • Are implemented as accessors (getters and setters)
  • May have (desirable) side effects
  • May set own members in other properties objects (besides grid.properties), such as those for columns, rows, and cells
