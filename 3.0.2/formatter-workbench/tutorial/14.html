Localizers tab

Binding a localizer to a cell

Once a localizer is registered in the grid, it can be bound to a cell (or column, etc.) through its format property.

Binding localizers declaratively

We have already seen how to do this using the state object (page 10).

Binding localizers programmatically

var propName = 'format'; // for example
var propValue = 'deg'; // for example

Example: Seting a grid property

grid.properties[propName] = propValue;

Example: Seting a column property

var columnName = 'prevclose'; // for example
var column = grid.behavior.columns[columnName];
column.properties[propName] = propValue;

Example: Seting a column property

var rowIndex = 3; // for example the fourth row
column.setCellProperty(rowIndex, propName, propValue);
Alternative, when you don't already have a column object handy: 
var columnIndex = grid.behavior.columns[columnName].index;
grid.behavior.setCellProperty(columnIndex, rowIndex, propName, propValue);
