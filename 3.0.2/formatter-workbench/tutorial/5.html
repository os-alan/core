Data tab

This tab provides an editor for the grid’s data source.
  • The data is stored as an array of dataRow objects.
  • The initial contents reflects the actual data shown in the grid instance above it.
  • After editing the data, bind it to the grid by clicking grid.setData(…).
  • The grid in turn is bound to the data window: Any edits made to cells (via cell editors) will alter the contents of this editor.
In actual practice, the data object passed to setData has typically been parsed from a character string in JSON format (by JSON.parse).

However, this editor uses eval to parse straight JavaScript, not JSON. When editing you do not need to concern yourself with strict adherence to the JSON standard (double quotes, primitive types, _etc._).
