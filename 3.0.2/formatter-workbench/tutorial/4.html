Editor pane

The four tabs in the lower left pane are for editing data and code.
  • Apply your edits by clicking the grey action button in the upper right of each tab.
  • The button’s label indicates the API call involved.
  • The API is called with the eval’d editor text as the first and only parameter (represented in the button label by an elipsis).

Synopsis

Data tab

The Data tab contains a JavaScript representation of the grid’s data source. This tab lets you edit the grid data to suit the needs of your localizer and cell editor development.

State tab

A Hypergrid state object contains grid property settings affecting look, feel, and behavior of a grid. This tab lets you edit a state object and load it into the grid.

Localizer tab

Hypergrid localizers control how strings are formatted for display. This tab lets you select an existing localizer for editing, or create and save* a new localizer.

Cell Editor tab

Hypergrid cell editors This tab lets you select an existing cell editor for editing, or create and save* a new localizer.
* This app persists data between sessions using LocalStorage only. To save your work per­ma­nent­ly, cut & paste to a file.
